# Cybersecurity Home Lab

A secure, privacy-focused home network designed to provide reliable household connectivity while serving as a practical cybersecurity learning environment.

## Goals

- Secure home networking
- Network segmentation
- VPN integration
- Infrastructure monitoring
- Virtualization and self-hosted services
- Cybersecurity skills development

## Current Infrastructure

### Firewall

- Protectli VP4650
- OPNsense

### Switching

- NETGEAR MS308E

### Wireless

- TP-Link Omada EAP610

### Virtualization

- Lenovo ThinkCentre M700 Tiny
- Proxmox VE

## Current Status

Operational:

- OPNsense deployed
- Internet connectivity operational
- Managed switching operational
- Wireless operational
- Proxmox deployed

In Progress:

- VLAN deployment
- WireGuard
- Mullvad VPN integration
- Pi-hole
- Grafana
