# Home Network Project

## Overview

This project is a secure, privacy-focused home network designed to provide reliable internet access for household devices while serving as a cybersecurity learning environment.

The project combines enterprise networking principles with practical home deployment requirements. Security, segmentation, monitoring, and documentation are prioritized while maintaining a stable and usable environment for daily family use.

The environment is intended to evolve over time through the addition of monitoring systems, virtualization services, remote access capabilities, and self-hosted applications.

---

# Primary Objectives

- Secure home network infrastructure
- Family internet access
- Privacy-focused networking
- Remote administration capability
- Network visibility and monitoring
- Cybersecurity learning platform

---

# Secondary Objectives

# Current Infrastructure

## Firewall

Protectli VP4650 running OPNsense

## Switching

NETGEAR MS308E Smart Managed Switch

## Wireless

TP-Link Omada EAP610

## Virtualization

Lenovo ThinkCentre M700 Tiny running Proxmox VE

## Power

CyberPower UPS with rack-mounted PDU
